# LLM Project

> **Demo 1.2.5** · 基于 Flutter 开发的本地 LLM 角色聊天 App  
> 当前名称为临时开发名，正式名称将在 Demo 1.3.0 阶段确定。

LLM Project 是一个面向 **LLM 角色聊天 / 角色卡创作 / 世界书设定 / 本地资产管理** 的 Android App。  
它不内置模型服务，也不托管用户内容；用户需要自行配置 API，并在本地创建、导入、导出自己的角色卡、世界书、背景等资产。

当前项目仍处于 **Demo 阶段**，部分 UI、数据结构、资产格式和功能逻辑仍可能在后续版本中调整。

---

## ✨ 当前主要功能

### 角色卡与系统卡

- 支持创建、编辑、删除角色卡
- 支持人物卡 / 系统卡两种类型
- 支持角色头像与 2:3 卡片封面
- 支持条目化角色设定
- 支持系统卡主角设定
- 支持多开场白管理与切换
- 支持角色切换轮盘
- 支持角色卡详情预览

### 世界书

- 支持世界书创建与编辑
- 支持世界书绑定到角色卡
- 支持关键词触发
- 支持常驻条目
- 支持递归触发
- 支持世界书 JSON 导入 / 导出
- 世界书卡片使用程序生成式封面，不依赖图片资产

### 背景系统

- 支持背景图库
- 支持图片背景
- 支持纯色 / 渐变背景
- 支持角色独立背景
- 支持背景卡导入 / 导出
- 图片背景导出时保留原图比例

### 聊天体验

- 支持 Markdown 消息显示
- 支持用户消息编辑
- 支持删除最近用户消息及其后续回复
- 支持 AI 回复重新生成
- 支持 AI 回复继续生成
- 支持开场白作为首条消息自动插入
- 支持当前卡用户设定覆盖
- 支持系统卡主角设定作为用户默认设定

### API 配置

- 支持多 API 配置
- 支持 Base URL、API Key、模型名配置
- 支持测试连接并获取模型列表
- API Key 使用 `flutter_secure_storage` 保存在本地

### 备份与资产导入导出

- 支持完整备份 `.llmbak`
- 支持角色卡文件包 `.llmcard`
- 支持角色卡图片 `.llmchar.png`
- 支持世界书 `.llmworld.json`
- 支持背景卡 `.llmbg.*`
- 支持导入前预览确认
- 支持内部资产标识校验
- 导出文件默认保存到 `Download / LLM Project /` 下的分类目录

## 🚀 快速开始

### 1. 安装 APK

下载 APK 后直接安装。  
如果系统提示未知来源应用，请在系统设置中允许当前浏览器或文件管理器安装应用。

### 2. 配置 API

进入：

```text
设置 → API 配置 → 添加配置
```

填写：

```text
配置名称
Base URL
API Key
模型名
```

配置完成后点击测试连接，成功后保存。

### 3. 创建角色卡

进入角色库，点击右上角 `+`：

```text
新建角色卡
导入角色卡
```

新建后点击角色卡进入编辑页。

### 4. 开始聊天

在角色库点击选中角色卡，再点击右下角播放按钮进入聊天页。

## 📦 资产格式说明

当前支持以下资产文件：

| 类型 | 后缀示例 | 用途 |
|---|---|---|
| 完整备份 | `.llmbak` | 迁移或恢复完整数据 |
| 完整角色卡 | `.llmcard` | 稳定分享或迁移单张角色卡 |
| 图片角色卡 | `.llmchar.png` | 适合展示和轻量分享 |
| 世界书 | `.llmworld.json` | 分享或迁移世界书 |
| 背景卡 | `.llmbg.png` / `.llmbg.jpg` | 分享或迁移图片背景 |

导入时不依赖文件后缀，而是读取文件内部的 LLM Project 资产标识。  
如果文件经过聊天软件压缩、转码或重新保存，内部数据可能丢失，导入时会失败。

### 图片角色卡注意事项

分享 `.llmchar.png` 时，请尽量选择：

```text
原图发送
或
以文件方式发送
```

如果直接通过聊天软件压缩发送，图片内部角色卡数据可能会被删除。

## 🔐 隐私说明

本 App 是本地工具：

- 不内置 API Key
- 不上传用户 API Key
- 不托管用户角色卡
- 不托管用户聊天记录
- 不提供官方内容社区

API Key 使用 `flutter_secure_storage` 保存在本地设备中。  
备份功能默认不导出 API Key，除非用户手动勾选。

```text

或

```

```text

重装 App
更换包名
```

