# Changelog

本文档记录 LLM Project 的主要版本变化。

---

## Demo 1.2.5

> 当前版本为 Demo 阶段测试版本，重点完善角色卡、世界书、背景、备份与资产导入导出能力。

### Added

#### 角色卡与系统卡

- 新增人物卡 / 系统卡类型区分
- 新增角色卡条目化编辑
- 新增系统卡主角设定
- 新增主角设定嵌套字段平铺编辑
- 新增角色头像与 2:3 卡片封面支持
- 新增角色切换轮盘
- 新增轮盘角色详情页
- 新增角色名底部胶囊入口
- 新增角色卡导入 / 导出能力
- 新增完整角色卡文件包 `.llmcard`
- 新增图片角色卡 `.llmchar.png`
- 支持角色卡导出时可选包含绑定世界书

#### 开场白

- 新增多开场白管理
- 新增开场白切换显示
- 新增无历史记录时自动插入首条开场白
- 修复新角色进入聊天页后开场白不显示的问题

#### 用户设定与主角设定

- 新增当前卡用户覆盖设定
- 新增全局用户设定作为默认昵称与头像
- 新增系统卡主角设定作为用户默认设定
- 新增“恢复默认设定”与“保存为角色卡默认”逻辑

#### 世界书

- 新增世界书管理
- 新增世界书关键词触发
- 新增常驻条目
- 新增递归触发
- 新增世界书导入 / 导出 `.llmworld.json`
- 新增世界书导入前预览确认
- 新增程序生成式世界书封面

#### 背景系统

- 新增背景图库
- 新增图片背景
- 新增纯色 / 渐变背景
- 新增角色独立背景
- 新增图片背景卡导入 / 导出
- 背景卡导出保留原图比例

#### 备份与恢复

- 新增完整备份 `.llmbak`
- 新增自定义导出内容
- 支持导出角色卡、世界书、背景、图片资源、用户设定、API 配置、聊天记录等
- 支持导入完整备份
- 支持备份导出到 `Download / LLM Project / Backups`
- 新增 API Key 和聊天记录敏感导出提醒

#### 资产系统

- 新增内部资产识别标识
- 导入时不依赖文件扩展名
- 新增导入前预览与完整性检查
- 新增角色卡、世界书、背景卡资产导入导出
- 新增导出到系统 Download 分类目录

#### Android 适配

- 新增网络权限配置
- 新增 `UCropActivity` 配置
- 新增状态栏 / 刘海屏适配
- 新增 Android MediaStore 下载目录保存逻辑

---

### Changed

- 优化角色卡编辑页布局
- 优化角色设定字段编辑方式
- 优化系统卡主角设定显示方式
- 优化聊天页角色名位置
- 优化角色轮盘交互入口
- 优化资源库排序交互
- 新建按钮合并为“新建 / 导入”入口
- 导出按钮合并为导出方式选择
- 角色库进入对话按钮移动到右下角
- 世界书库封面由灰色图标改为程序生成式封面
- 背景卡导出由固定比例改为原图比例
- 备份导出由自定义路径改为固定保存到 Download 分类目录

---

### Fixed

- 修复 Release APK 无法联网的问题
- 修复隐藏系统状态栏后顶部黑框问题
- 修复 image_cropper 选择图片后闪退问题
- 修复角色卡开场白首次进入不显示问题
- 修复角色卡编辑页多处像素溢出问题
- 修复角色库按创建时间排序不稳定问题
- 修复背景库 / 世界书库时间排序问题
- 修复 Android 公共目录写入权限导致备份导出失败的问题
- 修复导入 `.llmbak` 文件时系统文件选择器无法点击的问题

---

### Known Issues

- 当前仍为 Demo 阶段，软件名称、图标和部分 UI 仍可能调整
- 图片角色卡如果被聊天软件压缩，内部数据可能丢失
- 世界书高级触发机制尚未完善
- 状态栏功能目前仍处于基础占位阶段
- 输入功能区部分按钮仍为后续规划
- 角色心声 / 内心独白功能尚未实现
- 部分资产格式未来可能升级，但会尽量保持向后兼容
