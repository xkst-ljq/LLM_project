多文件补丁合并器

支持一次打多个文件的总补丁：新增 / 修改 / 删除。纯本地浏览器运行，文件不上传。
第一步：加载原项目文件夹
选你的项目根目录（比如 LLM_project 文件夹），浏览器会读取里面需要改的文件。不会上传。

第二步：粘贴总补丁
支持格式：
1. Multi-File SEARCH/REPLACE（推荐）
--- File: lib/pages/ui_studio_page.dart
<<<<<<< SEARCH
旧代码
=======
新代码
>>>>>>> REPLACE
--- File: lib/tools/new_helper.dart (new)
<<<<<<< SEARCH
=======
新文件全部内容
>>>>>>> REPLACE
--- File: lib/old_file.dart (delete)
可包含多个文件，一个接一个。
2. git unified diff（实验性）

结果
ZIP 里保持原目录结构，可直接解压覆盖到项目里。原文件不会被浏览器修改，放心。
